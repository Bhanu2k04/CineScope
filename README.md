## ✨ Features

### 🤖 AI-Powered Recommendations
- **Mood-Based ML Chatbot** — detects mood from natural language ("I feel adventurous", "something dark and gritty") using a trained multi-label classifier
- **Semantic Search** — sentence-transformers (all-MiniLM-L6-v2) find movies by vibe, not just keywords ("movies like Interstellar but sadder")
- **Collaborative Filtering** — learns from what users with similar taste enjoy
- **Content-Based Similarity** — genre Jaccard, director matching, cast overlap, rating proximity
- **Hybrid Engine** — all four signals merged and ranked per query

### 🎭 Discovery
- Browse by **18 genres** with color-coded cards
- Explore **6 languages** — English, Hindi, Telugu, Tamil, Malayalam, Kannada
- Decade browsing — 80s classics, 90s hits, 2000s blockbusters
- Trending, new releases, and free movies sections
- Fuzzy search with voice input support

### 👤 Profiles & Personalization
- Netflix-style **multi-profile** support per account
- Per-profile **watchlists**, genre/language **preferences**, like/dislike tracking
- **Mood history** — see your past chatbot moods in Settings
- Kids profile mode (filters to family-friendly content)

### 🎬 Movie Details
- Full cast and director cards with filmography links
- Embedded YouTube trailers
- Like / dislike / watchlist actions
- "More Like This" ML-powered similar movies
- Free movie player for public domain films

### 💬 CineBot Chatbot
- Resizable, draggable panel
- Voice input support
- Mood badge with confidence % displayed
- Multi-mood detection (e.g., "dark romantic thriller" → dark + romantic + suspenseful)
- Quick prompt buttons
- Feedback saved to database for future model improvement
- Graceful fallback to keyword matching if ML models not loaded

### 🎨 UI/UX
- Cinematic **landing page** with 3D poster grid and particle effects
- Animated splash screen with neon letter-by-letter reveal
- Framer Motion throughout — smooth transitions and hover effects
- Fully responsive — mobile to 4K
- Dark theme with purple/pink accent palette

## 🚀 Setup

### Prerequisites
- Python 3.10+
- Node.js 18+
- MongoDB 6.0+
- Git

### 1. Clone and install

```bash
git clone https://github.com/YOUR_USERNAME/cinescope.git
cd cinescope
```

**Backend:**
```bash
cd backend
python -m venv venv

# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
cp .env.example .env
# Edit .env with your values
```

**Frontend:**
```bash
cd ../frontend
npm install
```

### 2. Configure environment

Edit `backend/.env`:
```env
MONGO_URI=mongodb://localhost:27017/
DB_NAME=cinescopeDB
JWT_SECRET_KEY=your-secret-key-min-32-chars
ML_MODELS_DIR=./ml_models
```

### 3. Load data

```bash
cd backend
python -c "
from pymongo import MongoClient
import json
client = MongoClient('mongodb://localhost:27017/')
db = client['cinescopeDB']
with open('movies.json') as f:
    movies = json.load(f)
db.movies.drop()
db.movies.insert_many(movies if isinstance(movies, list) else [movies])
print('Movies loaded:', db.movies.count_documents({}))
with open('people.json') as f:
    people = json.load(f)
db.people.drop()
db.people.insert_many(people if isinstance(people, list) else [people])
print('People loaded:', db.people.count_documents({}))
"
```

### 4. Run

```bash
# Terminal 1 — Backend
cd backend && python app.py

# Terminal 2 — Frontend
cd frontend && npm start
```

App opens at **http://localhost:3000**

## 🧠 ML Training

Training runs in **Google Colab** (free GPU). Local CPU works but is slow.

### Steps

1. Open [Google Colab](https://colab.research.google.com)
2. Runtime → Change runtime → **GPU (T4)**
3. Upload `movies.json` to the Colab file browser
4. Open `backend/train_model.py` and paste each cell section into Colab
5. Run all cells — takes ~10 minutes with GPU

### What gets trained

| Model | Description | Output |
|---|---|---|
| Mood Classifier | Multi-label LogisticRegression, ~3000 synthetic samples | `mood_classifier.pkl` |
| Mood Vectorizer | TF-IDF (1-3 ngrams) for mood queries | `mood_vectorizer.pkl` |
| Label Encoder | MultiLabelBinarizer for 18 moods | `mood_label_encoder.pkl` |
| Movie Embeddings | sentence-transformers all-MiniLM-L6-v2 | `movie_embeddings.npy` |
| Movie IDs | Index mapping for embeddings | `movie_ids.npy` |
| TF-IDF Matrix | Bag-of-words fallback for movie search | `tfidf_matrix.npy` |
| TF-IDF Vectorizer | Movie corpus vectorizer | `tfidf_vectorizer.pkl` |

### Metrics you'll see

```
Mood Classifier Results:
  happy          | Accuracy: 0.923 | F1: 0.891
  sad            | Accuracy: 0.908 | F1: 0.874
  excited        | Accuracy: 0.934 | F1: 0.901
  ...
  Overall Accuracy (exact match): 0.847
  Macro F1:                       0.878
  Micro F1:                       0.891
```

Confusion matrices and F1 bar charts are saved as PNG files.

### After training

Download `ml_models.zip` from Colab and extract into `backend/ml_models/`.
Restart Flask — check `http://localhost:5000/health` for ML status:

```json
{
  "ml_classifier": true,
  "semantic_search": true,
  "tfidf_fallback": true,
  "mood_detection": "ml"
}
```

## 📡 API Docs

### Auth
| Method | Endpoint | Description |
|---|---|---|
| POST | `/auth/register` | Register new user |
| POST | `/auth/login` | Login, returns JWT |
| POST | `/auth/refresh` | Refresh access token |

### Movies
| Method | Endpoint | Description |
|---|---|---|
| GET | `/movies/popular` | Most voted movies |
| GET | `/movies/trending` | High-rated recent movies |
| GET | `/movies/new-releases` | Current year releases |
| GET | `/movies/free` | Movies with free stream URL |
| GET | `/movies/search?q=<query>` | Fuzzy title search |
| GET | `/movies/<id>` | Full movie details with cast |
| GET | `/movies/by-genre/<genre>` | Movies by genre |
| GET | `/movies/by-language/<lang>` | Movies by language |
| GET | `/movies/by-decade/<year>` | Movies by decade start year |
| GET | `/movies/by-decade-language/<year>/<lang>` | Decade + language filter |
| GET | `/movies/by-genre-language?genre=&language=` | Genre + language filter |

### People
| Method | Endpoint | Description |
|---|---|---|
| GET | `/people/popular` | Popular cast & crew |
| GET | `/people/search?q=<name>` | Search people by name |
| GET | `/people/<id>` | Person details + filmography |

### User
| Method | Endpoint | Description | Auth |
|---|---|---|---|
| GET | `/user/profile` | Get user profile | ✅ |
| PUT | `/user/profile` | Update profile | ✅ |
| PUT | `/user/profile/<idx>` | Update profile by index | ✅ |
| GET | `/user/profiles` | Get all profiles | ✅ |
| POST | `/user/profiles` | Create new profile | ✅ |
| PUT | `/user/preferences/<idx>` | Save genre/language prefs | ✅ |
| GET | `/user/watchlist/<idx>` | Get profile watchlist | ✅ |
| POST | `/user/watchlist/<idx>/<movie_id>` | Add to watchlist | ✅ |
| DELETE | `/user/watchlist/<idx>/<movie_id>` | Remove from watchlist | ✅ |
| POST | `/user/like/<movie_id>` | Like a movie | ✅ |
| POST | `/user/dislike/<movie_id>` | Dislike a movie | ✅ |
| GET | `/user/likes` | Get liked movies | ✅ |
| POST | `/user/reset-password` | Change password | ✅ |
| GET | `/user/mood-history` | Get chatbot mood history | ✅ |

### Recommendations
| Method | Endpoint | Description | Auth |
|---|---|---|---|
| POST | `/recommend/chat` | ML chatbot recommendations | Optional |
| GET | `/recommend/more-like-this/<id>` | Similar movies | Optional |
| POST | `/recommend/feedback` | Save chatbot feedback | Optional |
| GET | `/recommend/ml-status` | Check ML model status | ❌ |

```json
{
  "query": "I want something dark and romantic in Hindi",
  "user_id": "optional_user_id"
}
```

```json

  "message": "💕 Romantic mood · dark, Romance movies · in Hindi",
  "mood_result": {
    "primary_mood": "romantic",
    "primary_confidence": 0.847,
    "all_moods": [["romantic", 0.847], ["dark", 0.612]],
    "method": "ml",
    "emoji": "💕"
  },
  "found_genres": ["Romance", "Drama"],
  "found_languages": ["hi"],
  "found_people": []
}
```
